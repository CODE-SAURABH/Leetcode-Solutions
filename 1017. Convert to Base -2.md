# BruteForce Method

```python
class Solution:
    def baseNeg2(self, x: int) -> str:
        if x == 0: return "0"

        res = []
        while x:
            res.append(x & 1)
            x = -(x >> 1)
        
        return "".join(map(str, res[::-1]))
```

## Optimize Method

```python
class Solution:
    def baseNeg2(self, n: int) -> str:
        if n == 0:
            return "0"
        s=""
        while n != 0:
            rem=n%-2
            n//=-2
            if rem < 0:
                rem += 2
                n += 1
            s = str(rem) + s
        return s
```
