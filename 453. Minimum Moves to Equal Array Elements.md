# BruteForce Method

```python

class Solution:
    def minMoves(self, nums: List[int]) -> int:
        if len(set(nums)) == 1:
            return 0
        else:
            min_num = min(nums)
            moves = 0
            for num in nums:
                moves += num - min_num
            return moves


```

## Optimize Method

```python
class Solution:
    def minMoves(self, nums: List[int]) -> int:
        return sum(nums) - min(nums)*(len(nums))
```
