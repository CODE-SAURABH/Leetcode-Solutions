# BruteForce Method

```python
class Solution:
    def countBalls(self, lowLimit: int, highLimit: int) -> int:
        f={}
        for i in range(lowLimit,highLimit+1):
            no=sum(int(i) for i in str(i))
            if no not in f:
                f[no]=1
            else:
                f[no]+=1
        return max(f.values())
```

## Optimize Method

```python
class Solution:
    def countBalls(self, lowLimit: int, highLimit: int) -> int:
        map = [0]*46
        first = lowLimit
        index=0
        while first>0:
            index+=first%10
            first//=10
        map[index]+=1
        
        for i in range(lowLimit+1, highLimit+1):
            num=i
            while (num%10==0):
                index-=9
                num//=10
            index+=1
            map[index]+=1
            
        return max(map)
```
