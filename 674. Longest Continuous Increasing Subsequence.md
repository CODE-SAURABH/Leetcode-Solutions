
# Optimize Method

```python
class Solution:
    def findLengthOfLCIS(self, nums: List[int]) -> int:
        if len(nums)==1: return 1
        res = 0
        cons = 1
        for j in range(1, len(nums)): 
            if nums[j] > nums[j-1]:
                cons+=1
            else:
                res = max(res,cons)
                cons = 1
            # print(j,cons)
        return max(res,cons)

```
