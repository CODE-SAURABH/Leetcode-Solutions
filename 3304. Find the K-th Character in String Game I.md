# BruteForce Method

```python
class Solution:
    def kthCharacter(self, k: int) -> str:
        word = "a"
        alphabet = "abcdefghijklmnopqrstuvwxyz"
        
        while len(word) < k:
            new_part = ""
            for char in word:
                new_part += alphabet[(alphabet.index(char) + 1) % 26]
            word += new_part
        
        return word[k-1]


```

## Optimize Method

```python
class Solution:
    def kthCharacter(self, k: int) -> str:
        return ascii_lowercase[(k-1).bit_count()]
```
