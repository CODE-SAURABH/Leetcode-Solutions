# BruteForce Method

```python
class Solution:
    def countDistinctIntegers(self, nums: List[int]) -> int:
        f={}
        n=[int(str(i)[::-1]) for i in nums]
        for i in nums+n:
            if i not in f:
                f[i]=1
            else:
                f[i]+=1
        c=0
        for k,v in f.items():
                c+=1
        return c
```

## Optimize Method

```python
class Solution:
    def countDistinctIntegers(self, nums: List[int]) -> int:
        reversed_nums = [int(str(i)[::-1]) for i in nums]
        nums.extend(reversed_nums)
        return len(set(nums))
```

### OR

```python
class Solution:
    def countDistinctIntegers(self, nums: List[int]) -> int:
        seen=set()
        for i in nums:
            if i not in seen:
                seen.add(i)
            if int(str(i)[::-1]) not in seen:
                seen.add(int(str(i)[::-1]))
        return len(seen)
```
