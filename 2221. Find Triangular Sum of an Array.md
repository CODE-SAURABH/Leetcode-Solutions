# BruteForce Method

```python

class Solution:
    def triangularSum(self, nums: List[int]) -> int:
        while len(nums) > 1:
            nums = [(nums[i] + nums[i+1]) % 10 for i in range(len(nums) - 1)]
        return nums[0]

```

## Optimize Method

```python
import numpy as np

class Solution:
    def triangularSum(self, nums: List[int]) -> int:
        nums = np.array(nums)
        n = len(nums)
        while n > 1:
            nums[:-1] = (nums[:-1] + nums[1:]) % 10
            n -= 1
        return int(nums[0])

        
```
